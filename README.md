# Frontend Projects

This repository contains my frontend projects built with HTML, CSS, and JavaScript. I use it to practice web development, explore new ideas, and build interactive web applications.

## Projects

### 1. Kanban To-Do List

A simple Kanban-style task manager that lets you organize your tasks into different stages.

#### Features

* Create, edit, and delete tasks
* Add a title and description
* Drag and drop tasks between columns
* Organize tasks into:

  * To-Do
  * In Progress
  * Done
* Automatically saves tasks using Local Storage
* Displays the number of tasks in each column

#### Built With

* HTML
* CSS
* JavaScript
* Local Storage API

---

### 2. Image Editor

A simple image editor that allows you to apply filters, use built-in presets, and download the edited image directly from your browser.

#### Features

* Upload images from your device
* Adjust filters in real time
* Apply built-in filter presets
* Reset all filters with one click
* Download the edited image
* Responsive and clean interface

#### Built With

* HTML
* CSS
* JavaScript
* HTML5 Canvas API

## Getting Started

1. Clone this repository.

```bash
git clone https://github.com/your-username/frontend-projects.git
```

2. Open the project folder.

3. Open the `index.html` file of the project you want to run in your browser.
